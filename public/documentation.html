URL endpoints
Method Type URL Endpoint Parameters Required Request Response
GET '/users' None Return a list of all users and their data to the user returns Json data of all users
GET '/movies' None Return a list of all movies and their data to the user returns Json data of all movies
GET '/movies/:title' title Return a specific movie based on the title Returns Json data of a specific movie
GET '/director/:name' director name Return a specific director Returns Json data of a specific director
GET '/genre/:name' the genre name Return a specific genre Returns Json data of a specific genre
POST '/users' None Create a new user Pushes Json data of user to user's array
PUT '/users/:username' username Update user's name using username Replaces previous username with new one
POST '/users/:username/movies/:title' username and the movie title Post a movie in user's favorite list using username and movie title Pushes Json data selected movie to favoriteMovies array
DELETE '/users/:username/movies/:title' username and the movie title Delete specific movie using username and movie title from the user's favorite list Removes Json data of specific movie from favoriteMovie array
DELETE '/users/:username' username Delete user using username Removes Json data of user from the users array
