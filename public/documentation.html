GET '/movies/director/:directorName' director name Return a specific director Returns Json data of a specific director
GET '/movies/genre/:genreName' the genre name Return a specific genre Returns Json data of a specific genre

PUT '/users/:userId' user's id Update user's name using id Replaces previous username with new one
